Lilliana Checo
color language
In this experimental project, I explore both universal and personal meanings of color. Picking up a colored gel pen, I start to draw objects of the given color. However, the sequence of elements start forming a narrative as they evoke feelings within me, oftentimes making the composition more abstract. Ultimately, the page is full of symbols on which I print the most convinient emotion to describe it in Blackletter type calligraphy. The final composition is then traced in Illustrator to be easily reproducible on print. This is an ongoing project.
